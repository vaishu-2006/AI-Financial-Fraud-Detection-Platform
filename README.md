# 💳 AI Financial Fraud Detection Platform

An AI-powered Financial Fraud Detection Platform that leverages **Machine Learning**, **Explainable AI (XAI)**, and **Risk Analytics** to identify suspicious financial transactions in real time. The platform helps financial institutions detect fraud, analyze transaction patterns, visualize insights, and support intelligent decision-making.

> 🚧 **Project Status:** Under Active Development

---

## 📌 Overview

Financial fraud is one of the biggest challenges in the banking and fintech industry. This project aims to build an intelligent fraud detection system capable of classifying financial transactions as **Safe**, **Suspicious**, or **Fraudulent** using Machine Learning models.

The platform provides a modern dashboard, transaction management, fraud alerts, explainable AI, and comprehensive analytics to help analysts investigate fraudulent activities efficiently.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/vaishu-2006/AI-Financial-Fraud-Detection-Platform.git
```

## Navigate into Project

```bash
cd AI-Financial-Fraud-Detection-Platform
```

## Install Frontend Dependencies

```bash
npm install
```

## Start Frontend

```bash
npm run dev
```

Frontend runs at:

```
http://localhost:3000
```

---

## Backend Setup

```bash
cd backend
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Backend

```bash
uvicorn api.main:app --reload
```

Backend runs at

```
http://localhost:8000
```

API Documentation

```
http://localhost:8000/docs
```
